# YUMEBOOK
[![Image from Gyazo](https://i.gyazo.com/e930ff5a6df648b5dd67f237d9e3c644.png)](https://gyazo.com/e930ff5a6df648b5dd67f237d9e3c644)

## アプリURL
https://www.yume-book.com

## サービス概要
今日見た夢の内容をシェアするサービス。  
みんなの夢を覗き見たり、毎日の夢日記としても活用できます。
